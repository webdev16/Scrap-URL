# README

### Requirements

* Ruby version: 2.3.0

* Rails: >= 5.0.0

* Database: postgresql


## APIs

This supporting 3 apis

###### GET:  /pages 
list pages

###### POST: /pages
read pages and save to database

parameter:
  
* url: page URL to scrap, eg: https://www.google.com

###### GET:  /pages/:id    # get a specific page
